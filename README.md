DonorCapesAPI
=============

A Minecraft API for adding Donor/Tester only capes!

Use this to add tester or donor only capes to your mod!

Usage:

In your preInit method add this:

DonorCapesAPI.init(*CAPES TXT*, 
  			*DONOR CAPE URL*,
				*TESTER CAPE URL*);
